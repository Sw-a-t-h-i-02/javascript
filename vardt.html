Data Types

Primitive Datatypes
  1. Number

    2. Javascript has only one type of number. It can be decimal,floating etc., any kind of number is considered as a number in javascript



  2. String

    3. The Javascript strings are for storing and manipulating the text. The string can be written in single or double Quotation.Whatever the content you have written in quotations it will be considered as a string.



  3. Boolean

    4. The boolean in javascript works as a condition. It has only two value True or False.

Special Datatypes

There are two special datatypes:
  1. Undefined

    2. In javascript the undefined data type is considered when there is no value assigned to the variable.If the value is not assigned the it returs as undefined in the output.



  2. Null

    3. In javascript the null data type represents the absence of a value.It can be said as the variable has declared but the value is not assigned.
